Readme.md Goal Tracker Calendar A specialized web application designed to track personal goals through a visual, color-coded calendar interface. Inspired by habit-tracking visual methodologies. Core Features Visual Calendars: Up to 10 color-coded calendars stacked vertically. Dynamic Side Panel: Customizable position (Left/Right), containing goal summaries and time-linked notes. Bi-Directional Highlighting: Selecting a note highlights the day it was written. History Navigation: Easily jump between months and years. Google Auth: Secure login via Supabase. Tech Stack Frontend: HTML5, CSS3 (Tailwind CSS), Vanilla JavaScript. Backend/Database: Supabase (PostgreSQL). Authentication: Supabase Auth (Google OAuth). Hosting: Recommended: Vercel/Netlify. Implementation Journal Day 1: Project Architecture Defined the strategic plan in planning.md. Established the core UI philosophy: "Minimalist & Data-Driven". Decided on Vanilla JS + Supabase to minimize dependency rot and maximize AI readability. Project Structure /index.html: Main entry point. /css/style.css: Tailwind and custom styles. /js/app.js: Main application logic. /js/auth.js: Supabase Auth handlers. /js/db.js: Supabase database operations. Deployment This app is designed to be hosted on any static site hosting provider. Connect GitHub repo to Vercel. Add Supabase Environment Variables (SUPABASE_URL, SUPABASE_ANON_KEY). Deploy.
